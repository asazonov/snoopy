Load your files

Using the select button on the right, choose your BAM files, BAM index files and the text file containing the variants of interest. Multiple files can be loaded at once.

BAM/BAI files

The BAM files are expected to be indexed and to be named according to the convention that if reads.bam is the BAM file then reads.bam.bai is the index file.

Variant text files

Text files with the variants of interest expects one variant location per line with the locations having the following formats:

  • chr:position
  • chr-position
  • chr position
  • chr,position
